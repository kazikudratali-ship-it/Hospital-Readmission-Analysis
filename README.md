# 🏥 Hospital Readmission Analysis Dashboard

## 📌 Project Overview

This project analyzes **25,000+ hospital patient records** to identify patterns and risk factors associated with hospital readmissions, emergency visits, and patient outcomes.

The analysis was performed using **SQL for data analysis and transformation** and **Microsoft Power BI for interactive visualization and dashboard development**.

The objective is to help healthcare stakeholders understand readmission patterns across **age groups, medical specialties, diagnoses, diabetes status, emergency visits, and patient risk groups**.

---

## 🛠️ Tools & Technologies

* **SQL** – Data cleaning, transformation, aggregation, and exploratory analysis
* **Microsoft Power BI** – Interactive dashboard and data visualization
* **DAX** – KPI calculations and analytical measures
* **Power Query** – Data preparation and transformation

---

## 📊 Dashboard Overview

The Power BI dashboard provides insights into:

* Total Patient Count
* Readmission Rate
* Average Hospital Stay
* Average Number of Medications
* Average Lab Procedures
* Emergency Visit Rate
* Readmission Rate by Age Group
* Readmission Rate by Diagnosis
* Readmission Rate by Medical Specialty
* Readmission Rate by Diabetes Status
* Emergency Visit Rate by Age Group
* Patient Risk Group Analysis

---

## 🔍 Key Insights

* **Diabetes patients** show a higher readmission rate than non-diabetic patients.
* **Diabetes diagnosis** has the highest readmission rate among the analyzed diagnosis categories.
* **High-risk patients** have significantly higher readmission and emergency visit rates compared with medium- and low-risk groups.
* Readmission rates vary considerably across **age groups and medical specialties**.
